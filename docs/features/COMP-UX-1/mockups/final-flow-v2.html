CORE-LEARN-1 / design.md
Last modified: 3 minutes ago · 4 sessions · Claude

Self-Correction + Negative Feedback Architecture

The self-correction module enables SmartMemory to learn from negative feedback signals. When a retrieved memory is marked as unhelpful or incorrect, the system adjusts confidence scores and extraction weights to reduce similar false positives.

Feedback Signal Types

Three primary feedback channels: explicit user corrections via memory.correct(), implicit signals from re-retrieval patterns, and automated validation against ground truth during benchmark runs (ONTO-PUB-2).

Correction Pipeline

Corrections flow through a staged pipeline: signal intake, impact analysis, weight adjustment, and verification. The pipeline integrates with ONTO-PROV-1 provenance tracking to maintain an audit trail of all corrections.

Negative Weight Decay

Negative feedback applies exponential decay to affected entity confidence scores. Decay rate is configurable per-domain and respects SEC-DOMAIN-1 security boundaries.
